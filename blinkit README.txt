# Blinkit Sales Analysis Project

## Overview
This project involves deriving actionable insights from sales data of the grocery delivery company, Blinkit. Utilizing Excel, SQL, and Power BI, I created an interactive dashboard that highlights the best-performing sales regions, facilitating strategic decision-making.

## Features
- Data extraction and cleaning from the provided Excel file
- SQL queries to analyze and manipulate the data
- Interactive dashboard in Power BI showcasing key sales metrics
- Visualizations pinpointing top-performing sales regions
- Trend analysis to support strategic planning

## Tools and Technologies
- Excel for data extraction and cleaning
- SQL for data analysis and manipulation
- Power BI for creating interactive dashboards

Usage
Load the sales data Excel file into the data folder.

Execute the SQL queries to preprocess and analyze the data.

Import the analyzed data into Power BI.

Open the dashboard file in Power BI to interact with the visualizations.

Results
The interactive dashboard provides a comprehensive view of Blinkit's sales performance, highlighting regions with the highest sales and enabling data-driven decision-making.
